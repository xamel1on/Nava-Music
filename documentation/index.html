1.Login to your cpanel:
Open your browser type on your browser address bar http://www.yourdomain.com/cpanel

2.Type your user name and pass-word
Your host provides you with this information. If you use your own computer you may check the "remember my password", so you do not need to retype them the next time you visited. Hit ok.

3. File manager
Once you go inside the C-panel, you will find a lot of tools that you can use to control and manage your site. Look for FILE MANAGER. This is the area where you manage your files. Inside "file manager" there are a lot of different folder, each folder has there own purposes/use. For your visitor to be able to view your web pages you need to upload them to certain area/folder. That is public_html folderfolder Other host control panel might call the folder diffrently, your host supposes to tell you where the files should go.

4. Look for the public_html folder
Click the public_html folder icon, to open the folder. If you just sign up with your host most likely you will see a Cgi-bin folder and other folder, but it's depends some hosts give you more features in your hosting package.In this folder where we want to put our index.html and our flash movie file.

5.Upload the nava.zip file and extract it.

Now open http://www.yourdomain.com/index.html. You site will be live.

Please read this important note: Don't change the structure of the site.

Tutorial#back to top

Lets start updating the template. Open index.html file and follow the steps.

Most of the updates are same for all versions. So will explain once here.

Header - Logo
  1. Replace "images/logo-lite.png" with the your website logo url or source.

Navmenu

This is the basic structure.

    • SUB MENU
      • 
											
    • SUB MENU 1
      • 
										
    
									
    • 
									
  • MENU NAME 1
    •
  1. Replace "Menu name" with the menu name.
  2. Replace "MENU_LINK_HERE" with the menu link.
  3. Replace "SUB MENU" with the submenu menu name.
  4. Same for all. You can add unmited menus.

Contact Form

Open "contact.php" file from PHP folder and find this

							define( "RECIPIENT_NAME", "YOUR_NAME_HERE" );
							define( "RECIPIENT_EMAIL", "YOUR_EMAIL_HERE" );
  1. Replace "YOUR_NAME_HERE" with your name.
  2. Replace "YOUR_EMAIL_HERE" with your email where you want to receive the mail.

Footer Copyright
